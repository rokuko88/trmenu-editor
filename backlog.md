# TRMenu 编辑器功能规划

## 🔥 P0 - 核心功能（MVP 必需）

### 动作系统

- [x] 动作配置器界面（properties-panel 中的"配置动作"按钮）
- [x] 命令动作（COMMAND）配置
- [x] 菜单跳转（OPEN_MENU）配置
- [x] 关闭菜单（CLOSE）动作
- [x] 消息动作（MESSAGE）配置
- [x] 点击类型选择（左键/右键/Shift 等）
- [x] 动作优先级设置
- [x] 条件判断配置（require/deny）

### 导入导出

- [x] 导出为 TRMenu YAML 格式（完整配置）
- [x] 导入现有 TRMenu YAML 配置
- [x] YAML 语法验证
- [ ] 批量导出多个菜单
- [ ] 导出模板到资产库
- [ ] 导出预览功能

### 撤销重做

- [ ] 实现历史记录栈
- [ ] 撤销功能（Ctrl+Z）
- [ ] 重做功能（Ctrl+Y）
- [ ] 操作历史面板（可选）

### 右键菜单

- [x] 槽位右键 ContextMenu
- [x] 复制物品（Ctrl+C）
- [x] 粘贴物品（Ctrl+V）
- [x] 删除物品（Delete）
- [x] 克隆物品
- [x] 清空槽位
